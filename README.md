# Santhosh Babu Personal Blog

### Version
0.0.1

### Backend Technology Stack

* [nodeJS] 
* [jade] 
* [hapijs]
* [inert]


### Frontend Technology Stack

* [grunt]
* [bootstrap]


### Download application

```sh
$ git clone https://github.com/html5webster/santhoshbabu.git santhoshbabu
```

### Create Build
```sh
$ cd santhoshbabu
$ grunt
```

### Launch Server
```sh
$ cd santhoshbabu
$ npm start
```
