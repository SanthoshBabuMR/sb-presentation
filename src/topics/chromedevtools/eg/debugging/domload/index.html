Expected Outcome

  • DomContentLoaded should be fired, when HTML has been parsed and DOM tree is available.
  • JavaScript and CSS will also be loaded in memory, if this event has been fired

  • onLoad should be fired, when all resources including media (audio, video, image), font has been downloaded
