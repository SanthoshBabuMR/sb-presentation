JS - a functional language

  • 1. function can be passed as argument(s) to a function 
    function printThis() { console.log(this);}
function myFunction( printThis ) {
  // Do Something
}
  • 2. function can be returned from a function 
    function myFunction() {
  return function() {};
}

JS - a functional language

  • 3. function can be assigned to a variable 
    var myFun = function () {};
var myFun2 = function dummy() {};
// checkout the difference with <f_name>.name;
  • 4. function can be stored in an object or array 
    function printThis() { console.log(this);}
var myObj = {
	myFun : printThis;
}
  • 5. function can be invoked 
    function printThis() { console.log(this);}
printThis();

Function Literal

  • function <f_name> ( arg1, arg2, .. argn) {
  // statements/expression
}

Types of function invocation

  • 1. as a function invocation 
    eg: <f_name>( <args>);
  • 2. as a method call 
    eg: myObj.<f_name>( <args>);
  • 3. using apply() / call() 
    eg: myObj.<f_name>.call( <context>, <arg1>, .. , <argn>);
<f_name>.apply( <context>, [<arg1>, .. , <argn>]);
  • 4. as a constructor using new operator 
    eg: myObj = new <f_name>();

Callbacks

  • functions set to be invoked once the specified event has occured.
  • Browse Events
  • Network Events
  • DOM Events

arguments, this, length

  • arguments - retrieves the number of arguments passed to a function during runtime 
    
function sum(a,b) {
	console.log(arguments); // outputs [1,2,3]
	return a+b;
}
sum(1,2,3);

arguments, this, length

  • this - determines the context in which the function is being executed 
    
(function(){
  console.log(this); // outputs WINDOW
})()

arguments, this, length

  • <f_name>.length - specifies the number of params the function was original declared with 
    
function sum(a,b) {
	console.log(sum.length)
	return a+b;
}
sum(1,2,3);

Immediate, Anonymous, Closures

  • Immediate function - declared and invoked immediately 
    
(function sum(a,b) {
	console.log(sum.length)
	return a+b;
})(1,2,3)

Immediate, anonymous Closures

  • Anonymous function - declared with a function name 
    
(function(a,b) {
	console.log(sum.length)
	return a+b;
})(1,2,3);
