HTML Anchor Tag : <a>

Definition
The HTML Anchor tag is what makes the Web Exciting, enabling us to surf from one page to another page. The page may be a text document, audio, video, image or any resource that the browser can render itself or make use of a plugin to render it.
  • <a> tag creates a hyperlink
  • Used for creating hyperlinks to another document
  • Used for linking to different sections within the same document, which is referred to as bookmark

Example
Linking to Other Documents:

<a href="http://www.google.com/" target="_blank">Google</a>

Linking to different sections within the same document

<a name="top">Move to Top</a>

Usage
Using padding and margins with anchor tag can be quite handy.
for example,

<a style="margin:5px; padding:5px; background:#66F; display:block; color:#FFF; width:46px;" href="http://www.google.com">Google</a>

will give a fine looking link : Google

Pseudo-class
Generally anchor elements, appear in blue color with an underline, if no CSS is applied to them.
  • New Link: Yahoo.com

Once user clicks on a link or comes across an already visited link, it appears in purple color.

  • Visited Link: Yahoo.com

Anchor tag has different link states, which is rendered in a browser using Pseudo-Classes ,

pseudo-classes Description
a:link The normal state of a link or simply an unvisted link.
a:visited The state of a visited link.
a:hover The state of the link, when a user hovers the mouse pointer on top of the link.
a:focus The state of the link, when it gains focus.
i.e., When a user uses the keyboard "TAB" key to navigate through the links in a page.
a:active The state of the link, when a user precisely clicks on the link using the mouse or hits enter key.
